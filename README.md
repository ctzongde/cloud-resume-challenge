# AWS Cloud Resume Challenge

A cloud-based resume website built as part of the Cloud Resume Challenge.  This project demonstrates practical AWS skills by deploying a static resume  as a serverless web application.

# Tech Stack:
- Frontend: HTML, CSS, JavaScript
- Hosting: Amazon S3, CloudFront
- Backend: API Gateway, Lambda (Python), DynamoDB
- Iac: Terraform
- CI/CD: GitHub Actions
